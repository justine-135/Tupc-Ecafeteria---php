# Tupc-Ecafeteria

Tupc-Ecafeteria is an online canteen system where vendors can upload their food menu and display it to the customers. Vendors can also keep track of their sales using an inventory system.


Tech used:
    HTML
    CSS
    Bootstrap
    Javascript
    JQuery
    PHP
    MySQL
